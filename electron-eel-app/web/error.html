Erro ao Iniciar Aplicativo

⚠️

Não foi possível iniciar o servidor Python necessário para o funcionamento do aplicativo.

Dicas para resolver o problema:
  • Verifique se o Python 3 está instalado em seu sistema
  • Certifique-se de que as bibliotecas necessárias (eel, psutil) estão instaladas
  • Verifique se a porta 8080 não está sendo usada por outra aplicação

Para mais detalhes, verifique o console de desenvolvimento ou logs do aplicativo.
